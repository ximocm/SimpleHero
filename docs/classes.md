# Classes - Base Doc

## Purpose
Define playable classes with clear combat roles and simple progression.

## Rules
- 3 starting classes for MVP.
- Each class starts with 1 active skill.
- Magic is exclusive to Mage for now.

## Class Definitions (MVP)

### Warrior
- Role: Frontline melee.
- Preferred gear: Heavy Armor, Shield, Short Sword, Two-Handed Sword.
- Skill: `Power Strike`
- Effect: Next basic attack this turn deals `+3` damage.
- Cooldown: 2 turns.

### Ranger
- Role: Mobile physical damage.
- Preferred gear: Light Armor, Bow.
- Skill: `Dash`
- Effect: Move up to `+2` extra tiles this turn.
- Cooldown: 2 turns.

### Mage
- Role: Utility and spell damage.
- Preferred gear: Light Armor, Staff.
- Skill: `Spellcast` (magic-only class action)
- Effect: Cast one spell per turn:
- `Heal`: Restore HP to one ally.
- `Fireball`: Area magic damage.
- `Ice`: Control spell (slow or root one enemy for 1 turn).
