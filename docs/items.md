# Items - Base Doc

## Purpose
Define MVP item stats in the simplest possible way.

## Rules
- Weapons only use: `Range`, `Attacks (d6)`, `Damage`.
- Weapon effects are disabled for now, except `Channel Magic`.
- Armor table only uses: `Name`, `Reduction`.
- Light Armor gives `+1 Movement`.
